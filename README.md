# ClawPen Website

React-based website for ClawPen.ca

## Tech Stack
- React 18
- Vite (fast build tool)
- Tailwind CSS (styling)
- React Router (multi-page ready)

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import to Vercel
3. Connect clawpen.ca domain
4. Done

## Structure
```
src/
├── components/     # Reusable components
├── pages/         # Page components
├── App.jsx        # Main app with routing
└── main.jsx       # Entry point
```
